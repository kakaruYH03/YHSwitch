# YHSwitch

<br/>

#### uGUI를 이용한 Switch
<br/>

유니티에서 사용할수있는 스위치

1. 스위치 색상변경
- 유니티 내에서 스위치의 배경색과 핸들색 변경 기능
- 버튼 클릭시 핸들 속도에 따라 색상이 바뀜

2. 스위치 핸들 이동
- 클릭 했을때 스위치가 부드러운 애니메이션으로 이동함
- 유니티 내에서 애니메이션의 속도 조절 가능
- 연속 클릭해도 고장안남
